# Neverness To Everness (NTE) ID System

**Game Code**: `7`

---

## ID 구조 (10자리)

모든 ID는 **10자리 숫자**로 통일됩니다.

### 형식: `GC TT YY NNNN`

| 위치 | 설명 | 예시 |
|------|------|------|
| GC (2자리) | 게임코드 + 카테고리 | 72 = 캐릭터 |
| TT (2자리) | 타입/등급 | 04 = A급, 06 = S급 |
| YY (2자리) | 서브분류 | 01 = 첫번째 속성 |
| NNNN (4자리) | 순번 | 0001~9999 |

---

## 1. 캐릭터 ID: `72 RR EE NNNN`

- **72**: 캐릭터 카테고리
- **RR**: 등급 (04=A급, 06=S급)
- **EE**: 속성 코드
- **NNNN**: 순번

### 속성 코드

| 코드 | 속성 ID | 한국어 | 영어 | 색상 |
|------|---------|--------|------|------|
| 01 | incantation | 주 | Incantation | #E04778 |
| 02 | anima | 령 | Anima | #38C297 |
| 03 | cosmos | 빛 | Cosmos | #D2CCA9 |
| 04 | psyche | 혼 | Psyche | #6A71F4 |
| 05 | lakshana | 상 | Lakshana | #DDCB68 |
| 06 | chaos | 암 | Chaos | #A36EDB |

### 예시
```
7204010001 - 첫번째 A급 Incantation(주) 캐릭터
7206020001 - 첫번째 S급 Anima(령) 캐릭터
7204040001 - 첫번째 A급 Psyche(혼) 캐릭터
```

---

## 2. 무기 ID: `73 RR TT NNNN`

- **73**: 무기 카테고리
- **RR**: 등급 (03=B급, 04=A급, 06=S급)
- **TT**: 무기 타입 코드
- **NNNN**: 순번

### 무기 타입 코드

| 코드 | 타입 ID | 이름 |
|------|---------|------|
| 01 | ark | Ark (아크) |

### 예시
```
7306010001 - 첫번째 S급 Ark 무기
7304010001 - 첫번째 A급 Ark 무기
7303010001 - 첫번째 B급 Ark 무기
```

---

## 3. 재료 ID: `71 CC SS NNNN`

- **71**: 재료 카테고리
- **CC**: 재료 종류
- **SS**: 서브 카테고리 (그룹 번호)
- **NNNN**: 순번 또는 티어

### 재료 종류 코드 (CC)

| 코드 | 카테고리 | 설명 | 티어 |
|------|----------|------|------|
| 00 | credit | 게임 재화 | 없음 |
| 10 | common | 공통 승급 재료 | 3티어 |
| 20 | forgery | 스킬 재료 | 3티어 |
| 30 | ascension | 캐릭터별 승급 재료 | 없음 |
| 40 | boss | 보스 재료 | 없음 |
| 50 | weeklyBoss | 주간 보스 재료 | 없음 |
| 60 | player_exp | 캐릭터 경험치 ⭐ | 없음 |
| 70 | weapon_exp | 무기 경험치 ⭐ | 없음 |

⭐ = EXP 카테고리 (value 필드 필수, 자동 인식)

### 티어 규칙 (NNNN)

**티어 있는 재료 (common, forgery)**: 0001=T1, 0002=T2, 0003=T3
**티어 없는 재료**: 0001, 0002, 0003... 순차

### 예시
```
7100000001 - 게임 재화 (credit)
7110010001 - Common 첫번째그룹 T1
7110010002 - Common 첫번째그룹 T2
7110010003 - Common 첫번째그룹 T3
7110020001 - Common 두번째그룹 T1
7120010001 - Forgery 첫번째그룹 T1
7130010001 - 첫번째 캐릭터 ascension 재료
7140010001 - 첫번째 boss 재료
7150010001 - 첫번째 weeklyBoss 재료
7160010001 - player_exp 아이템 1번
7170010001 - weapon_exp 아이템 1번
```
